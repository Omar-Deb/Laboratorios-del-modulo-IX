### IV. Configuración del Cliente Windows (VM W11)
wf.msc
Comentario: Permite habilitar gráficamente las reglas de entrada de "Administración remota de Windows" en el Firewall.
________________


Script de PowerShell (Ejecutar como Administrador)
Configuración:
PowerShell
# 1. Políticas de ejecución y PSRemoting
Set-ExecutionPolicy RemoteSigned -Force
Enable-PSRemoting -Force


# 2. Configurar WinRM para Ansible
Set-Item WSMan:\localhost\Service\AllowUnencrypted -Value $true
Set-Item WSMan:\localhost\Service\Auth\Basic -Value $true
Set-Item WSMan:\localhost\Client\TrustedHosts -Value "*" -Force


# 3. Firewall y WinRM
netsh advfirewall firewall add rule name="WinRM HTTP" dir=in action=allow protocol=TCP localport=5985
netsh advfirewall firewall set rule group="Administración remota de Windows" new enable=yes


# 4. Crear Usuario y Privilegios
Restart-Service WinRM
$Password = ConvertTo-SecureString "1234" -AsPlainText -Force
New-LocalUser -Name "ansible" -Password $Password -FullName "Ansible User"
Add-LocalGroupMember -Group "Administradores" -Member "ansible"


Comentario: Habilita el protocolo WinRM, configura la autenticación básica, abre los puertos necesarios en el firewall y crea el usuario administrador para la gestión remota.
________________


winrm enumerate winrm/config/listener
Comentario: Comprueba que el servicio de escucha de WinRM esté activo en el equipo Windows.
________________


Test-NetConnection -ComputerName localhost -Port 5985
Comentario: Verifica localmente que el puerto de administración remota esté abierto y listo para recibir conexiones.

PRÁCTICA 5: Automatización con Playbooks
sudo nano /etc/ansible/install_notepad.yml Configuración:
YAML
- name: Descargar e instalar Notepad++
  hosts: win
  tasks:
    - name: Descargar el instalador
      win_get_url:
        url: "https://github.com/notepad-plus-plus/notepad-plus-plus/releases/download/v8.5.8/npp.8.5.8.Installer.x64.exe"
        dest: 'C:\Users\ansible\Downloads\npp_installer.exe'
    - name: Instalar de forma silenciosa
      win_shell: |
        Start-Process -FilePath "C:\Users\ansible\Downloads\npp_installer.exe" -ArgumentList "/S" -Wait
      args:
        executable: powershell


Comentario: Playbook que descarga e instala Notepad++ en Windows de manera desatendida.
________________


sudo nano /etc/ansible/install_nginx.yml Configuración:
YAML
- name: Instalar y configurar Nginx en Linux
  hosts: linux
  become: yes
  tasks:
    - name: Instalar Nginx
      apt:
        name: nginx
        state: latest
        update_cache: yes
    - name: Habilitar servicio
      service:
        name: nginx
        state: started
        enabled: yes

$Password = ConvertTo-SecureString "1234" -AsPlainText -Force
